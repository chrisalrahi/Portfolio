Beer Goggles

makerepo.com/ChristopherAlRahi/beergoggles

Beer Goggles is a gravity and temperature monitoring system designed for Beyond the Pale Brewing. The prototype combines a load cell, ultrasonic sensor, and Arduino control to automate data collection during fermentation. I handled both the mechanical integration and sensor calibration, using 3D printing, wiring, and custom code to create a compact, closed-loop system that delivers accurate, real-time readings.

Beyond the Pale is a local brewing company in Ottawa that needed a better way to measure specific gravity and temperature during beer production. Tracking these values is a key part of the brewing process, but their existing method was slow and inconsistent, leading to wasted time and product. As the company plans to scale up operations, improving this system became essential.

A specific gravity and temperature monitoring system is needed to streamline and support Beyond the Pale Brewingâ€™s production as they expand. The system should enable simple, accurate, and continuous data collection directly from the brewing tanks. It must operate reliably under brewing conditions, maintain hygienic standards, and deliver consistent, real-time measurements to improve efficiency and product quality.

The final design uses a load cell and ultrasonic sensor connected to an Arduino Uno to calculate specific gravity automatically. A peristaltic pump transfers the beer between the main tank and a sealed side reservoir, while a custom program collects and visualizes real-time data. The result is a clean, closed-loop system that gives brewers consistent, accurate measurements with minimal manual input.
